# Deployment Commands

## 1. Deploy Backend to Render

```bash
# Commit and push changes to trigger Render deployment
git add .
git commit -m "Fix backend configuration for Render deployment"
git push origin master
```

## 2. Update Vercel Environment Variables

Run these commands to update Vercel environment variables:

```bash
# Update backend URL to point to Render
vercel env add NEXT_PUBLIC_BACKEND_URL production
# Value: https://autoria-backend.onrender.com

# Update NextAuth URL
vercel env add NEXTAUTH_URL production  
# Value: https://autoria-clone.vercel.app

# Update Google OAuth settings (if needed)
vercel env add GOOGLE_CLIENT_SECRET production
# Value: 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

# Update NextAuth Secret (if needed)
vercel env add NEXTAUTH_SECRET production
# Value: [generate a new secret]
```

## 3. Redeploy Frontend

```bash
# Trigger new Vercel deployment with updated environment variables
vercel --prod
```

## 4. Test Deployment

After deployment, test these URLs:
- Backend health: https://autoria-backend.onrender.com/health/
- Frontend: https://autoria-clone.vercel.app
- API docs: https://autoria-backend.onrender.com/api/doc/
- Admin: https://autoria-backend.onrender.com/admin/
